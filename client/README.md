# AWK IDE

VSCode client extension for AWK Language Server.

## Features

- [x] Syntax highlighting
- [x] Diagnostics
- [x] Autocomplete
  - [x] Builtins
  - [ ] User defined symbols
- [x] Hints on hover
  - [x] Builtins
  - [ ] User defined symbols
- [x] Go to definition
- [x] Code outline & symbol references
- [x] Document symbols
- [x] Workspace symbols
- [ ] Rename symbols
- [ ] Code formatting
